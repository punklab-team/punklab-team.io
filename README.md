# Описание

Документация для техниеской команды Punklab

## Разработка

### У становка зависимостей

``` shell
pip install poerty
poery install
```

### Запуск проекта

``` shell
mkdocs serve
```

## Деплой

``` shell
mkdocs build
```

После выполнения команды появится папка **site** с файлом **index.html**
